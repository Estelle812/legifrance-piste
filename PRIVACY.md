# Privacy Policy — Légifrance PISTE Plugin

**Last updated: June 2026**

## Data collection

This plugin does not collect, store, or transmit any personal data.

## How it works

- Your PISTE API credentials (`client_id` and `client_secret`) are stored **locally** on your machine in Claude Desktop's configuration file. They are never sent to any third party other than the official PISTE gateway (`oauth.piste.gouv.fr` and `api.piste.gouv.fr`).
- Search queries are sent directly from your machine to the Légifrance API via PISTE. They are not logged or stored by this plugin.

## Third-party services

This plugin connects to:
- [PISTE](https://piste.gouv.fr) — the French government's API gateway, operated by AIFE
- [Légifrance](https://www.legifrance.gouv.fr) — operated by DILA

Please refer to their respective privacy policies for information on how they handle requests.
