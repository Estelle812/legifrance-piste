## Correspondances thématiques rapides

- **Contrat, responsabilité civile, propriété** → Code civil
- **Emploi, licenciement, congés, syndicats** → Code du travail
- **Sociétés, faillite, concurrence** → Code de commerce
- **Infractions, peines** → Code pénal
- **Procédure judiciaire civile** → Code de procédure civile
- **TVA, impôt sur le revenu, IS** → Code général des impôts
- **Retraite, maladie, chômage** → Code de la sécurité sociale
- **Protection du consommateur** → Code de la consommation
